# 维多利亚州交通事故严重程度预测与风险场景识别

本项目利用机器学习技术分析维多利亚州道路碰撞数据，旨在预测事故严重程度并识别高风险事故场景。

## 研究问题

1. **严重程度预测**：能否利用机器学习模型准确预测交通事故的严重程度？
2. **风险场景识别**：能否通过聚类技术识别出高风险的交通事故模式？



## 数据集

数据来源于 [维多利亚州政府数据目录](https://discover.data.vic.gov.au/dataset/victoria-road-crash-data)，整合了警方报告和医院伤亡信息，涵盖了时间、地点、环境条件、事故类型及道路使用者等维度。

## 项目流程

### 1. 数据预处理

- **数据整合**：合并车辆 (`filtered_vehicle.csv`)、事故 (`accident.csv`) 和人员 (`person.csv`) 原始数据集。
- **清洗与编码**：在 `fillandcode.ipynb` 中完成缺失值处理及特征编码。
- **聚合**：通过 `combine.ipynb` 生成用于建模的最终聚合数据集 `final_aggregated_data.csv`。

### 2. 无监督学习：聚类分析 (`Clustering.ipynb`)

- 使用 **K-Means** 算法对事故特征进行聚类。
- 利用 **肘部法 (Elbow Method)** 确定最佳聚类数（K=3）。
- 通过 **PCA (主成分分析)** 进行维度压缩和可视化，识别不同风险等级的场景。

### 3. 有监督学习：严重程度预测 (`severity_prediction.ipynb`)

- **模型训练**：对比了 KNN、随机森林 (Random Forest) 和 XGBoost 模型。
- **特征筛选**：对比了 5、10 和 20 个特征的建模效果，最终确定 20 个最优特征。
- **性能评估**：使用 Macro Recall 作为主要指标，并重点关注 Class 0（致命事故）的召回率。最终实验表明 **随机森林** 为最优模型。
